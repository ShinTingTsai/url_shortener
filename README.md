# URL Shortener



Demo:
https://gentle-falls-00001.herokuapp.com

## 功能

1. Shorten URL
2. Copy shortened URL by one-click

## 環境建置與需求
- Node.js v10.15.0
- Express v4.17.1
- express-handlebars: 5.1.0
- body-parser: 1.19.0
- mongoose: 5.9.27

## 安裝與執行步驟
- 下載專案到本機
```
git clone https://github.com/ShinTingTsai/url_shortener.git
```
- 切至專案資料夾
```
cd url_shortener
```
- 安裝套件
```
npm install
```
- 開啟程式
```
npm run dev
```
- 請至http://localhost:3000開始使用程式
